# Introduction

Pampalo is a next generation personal finance application that is privacy by default.

Great power cuts both ways. Since the widespread adoption of AI, attackers have moved faster: vulnerability disclosures hit record highs, the window between CVE publication and active exploitation has collapsed to days, and AI-assisted phishing and deepfake fraud have grown by orders of magnitude. AI didn't create these trends, but it is accelerating them.

Removing honey pots (attractive data stores to raid) is an important design decision to factor in moving forward, and is Pampalo's northern star.
