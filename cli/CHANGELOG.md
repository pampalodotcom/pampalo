# @pampalo/cli

## 0.1.0

### Minor Changes

- Initial public release: headless Pampalo agent accounts (SDK + CLI).
  - `@pampalo/shared` — protocol crypto: notes, ZK proofs, ECIES, Poseidon merkle tree.
  - `@pampalo/sdk` — agent-account custody, transport, SQLite note store, sync, and
    shield/transfer/unshield/send (proofs generated under Node).
  - `@pampalo/cli` — the `pampalo` command-line tool.
